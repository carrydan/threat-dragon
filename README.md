## CI/CD Pipeline

Автоматизированный процесс сборки и деплоя реализован с помощью GitHub Actions. Пайплайн включает следующие этапы:

- Установка зависимостей.
- Запуск тестов.
- Сборка приложения.
- Сборка и публикация Docker-образа.
- Деплой в Kubernetes-кластер.

Файл конфигурации пайплайна находится по пути `.github/workflows/ci-cd.yml`.
